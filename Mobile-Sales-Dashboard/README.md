# 📱 Mobile Sales Dashboard

## Overview
This Power BI dashboard analyzes mobile phone sales performance across different cities, brands, payment methods, and customer ratings. The dashboard provides actionable insights for sales and business decision-making.

## Dashboard Features
- Total Sales Tracking
- Quantity Sold Analysis
- Transaction Monitoring
- Average Sales Calculation
- City-wise Sales Performance
- Brand-wise Sales Analysis
- Monthly Sales Trends
- Customer Ratings Analysis
- Payment Method Distribution

## Key Metrics
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales Value

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## Business Insights
- Identified top-performing cities.
- Compared sales performance across mobile brands.
- Analyzed customer satisfaction through ratings.
- Evaluated preferred payment methods.
- Tracked monthly sales trends and seasonality.
